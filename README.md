# osupacks

osu\![官方曲包](https://osu.ppy.sh/beatmaps/packs)收录列表，这意味着曲包中皆为被官方认可的谱面。曲包以谱面打包成压缩包的形式存在，而收录列表记录着谱面的名称、收录时间、大小等以供玩家参考。
